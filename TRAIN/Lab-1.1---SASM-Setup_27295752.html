Training2023 : Lab 1.1 - SASM Setup

Created by Vishal Thakur, last modified on May 30, 2023

Part 1

Download SASM

Step 1

Option 1: Google “github pages sasm” and go to the official GitHub page for SASM:

Option2: Go to this link directly:

https://dman95.github.io/SASM/english.html

Step 2

On the GitHub page, download the SASM version “Download .exe for Windows”, as shown in the image below:

Part 2

Install SASM

Step 1

Once the executable has been downloaded, double-click on it and it should launch the installer for Windows. ‘Accept’ the agreement when ready and click on ‘Next’.

Step 2

Go through the installation process, create a desktop shortcut as it will come handy during the workshop.

Step 3

On the last screen of the installation, you can check the box to ‘Launch SASM’ and click ‘Finish’. This should start the SASM application once the installation is complete.

SASM is now installed.

Part 3

Initial configuration

Step 1

Even though you selected a language earlier during the installation process, the following prompt is the first thing you see once you launch the application. Its very important that you change the language to your preferred language at this prompt.

Step 2

From the main menu at the top, click on ‘Settings’.

On the Settings screen, click on the ‘Build' tab and change the Mode to ‘x64’.

If needed, change the Assembler to ‘NASM’

SASM is now installed, configured and ready to use for the workshop!

Please make sure that this lab is completed before you start the rest of the workshop. If there are any issues, please raise them to the instructor.
